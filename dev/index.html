SpinWaveTheory

Introduction

Spin wave theory for magnetically ordered quantum lattice systems based on the QuantumLattices pack and the TightBindingApproximation pack.

Installation

In Julia v1.6+, please type ] in the REPL to use the package mode, then type this command:

pkg> add SpinWaveTheory
